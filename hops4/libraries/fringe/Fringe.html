FringeÔÉÅ

The Fringe library provides a algorithmic management framework for Very Long Baseline Interferometry (VLBI) fringe fitting and data processing. This library implements the core fringe fitting algorithms, data configuration, data summarization and visualization capabilities required for VLBI data analysis.
