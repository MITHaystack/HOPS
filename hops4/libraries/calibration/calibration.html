MHO_MixedPolYShift

Class

MHO_MixedPolYShift

Operator Type

Unary

Operator Category

calibration

Argument Data Type

visibility_type

Priority Value

3.5

Control File Keyword

mixed_pol_y_shift

Applies a 90 degree phase offset to the Y-polarization of each station which the linear polarization member of a mixed linear-circular polarization baseline. Intended for use in mixed S/X-VGOS geodetic experiments.

MHO_PolProductSummation

Class

MHO_PolProductSummation

Operator Type

Unary

Operator Category

calibration

Argument Data Type

visibility_type

Priority Value

3.99

Performs polarization product summation with support for parallactic angle corrections for both reference and remote stations to form the pseudo Stokes-I polarization product (VGOS geodesy). Triggered by the command line syntax -P I.
