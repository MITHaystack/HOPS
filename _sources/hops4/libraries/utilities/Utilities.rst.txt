.. _UtilitiesLibrary:

Utilities
=========

The utilities library provides essential support classes and functions for the 
HOPS4 framework. This library offers a set of tools for time/date handling, 
file operations, meta-programming utilities, legacy (HOPS3) compatibility, 
and other miscellaneous support functions. See the respective sections below 
for more details.
