.. _MathLibrary:

Math
====

The Math library provides basic mathematical functions and algorithms for 
the HOPS4 framework. This library offers a set of tools for 
Fast Fourier Transform operations, and other numerical utilities required 
by the fringe fitting algorithm and other VLBI data processing applications.
